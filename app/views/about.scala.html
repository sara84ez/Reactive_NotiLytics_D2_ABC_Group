About NotiLytics

Project Overview

NotiLytics is a real-time, actor-based, reactive news streaming platform developed for Delivery 2 of SOEN 6441 (Software Design Methodologies). This version of the system replaces traditional HTTP request-response patterns with a pure WebSocket architecture powered by typed Pekko actors.

The system supports real-time article streaming, live search functionality, News API source exploration, and an actor-supervised backend designed for scalability, concurrency, and fault tolerance.

Technologies Used

  • Pekko Typed Actors
  • Reactive WebSocket Communication
  • Play Framework
  • Bootstrap 5 Dark Theme
  • NewsAPI Integration
